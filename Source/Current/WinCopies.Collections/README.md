Description
-----------

Some utilities to extend the .NET Framework capabilities, such as:

- An array and list builder;
- (ReadOnly)Collections;
- (ReadOnly)(Observable)Stack/Queue/LinkedCollections;
- Enumerables;
- Enumerators
